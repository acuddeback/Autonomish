This is a sample Blog Post

I can't currently afford to get a nice blog plugin so, for the time being, this is just being done in HTML which needs a little help. Especially the sticky navbar and the way it cuts off the top of the page when it sticks. it's okay for now, though. I just have to go read through a bunch of documentation on overriding bootstrap style and script.

Fortunately, this blog is run by a few tech savvy teenagers, so we can make this work.

I may decide to integrate Blogger into this site using jquery or something, but for now, this works as a proof of concept.

We--Ilana and I--hope to use this blog to write commentary on tech issues that we find interesting in a way that's accessible to the common man. One of the biggest problems with the rise of AI is that the programmers, policymakers, and consumers don't speak the same language. Thus, having productive discussions about AI ethics can be incredibly challenging. This is our way of hopefully making a contribution to those discussions in a way that will allow more people to participate.

Cheers,

Anna
